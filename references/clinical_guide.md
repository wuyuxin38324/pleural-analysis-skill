# 胸膜侵犯临床参考指南

## 背景

胸膜侵犯(Pleural Invasion, PI)是非小细胞肺癌的重要预后因素,影响手术方式和生存率。

## 诊断标准

根据IASLC(国际肺癌研究协会)分类:

- **PL0**: 肿瘤未达脏层胸膜表面
- **PL1**: 肿瘤穿透脏层胸膜弹性层,但未达表面
- **PL2**: 肿瘤穿透脏层胸膜表面
- **PL3**: 肿瘤侵犯壁层胸膜

## 影像学危险因素

### CT表现
1. 胸膜牵拉征
2. 胸膜接触距离 < 5mm
3. 肿瘤最大径 > 20mm
4. 实性成分比例 > 50%

### PET表现
1. SUVmax > 5 (提示高代谢,侵袭性强)

## Kong et al. 2025 模型简化评分

| 因素 | 分值 |
|------|------|
| SUVmax > 8 | +30 |
| SUVmax 5-8 | +20 |
| SUVmax 3-5 | +10 |
| 胸膜接触(距离<0) | +40 |
| 距离 0-5mm | +30 |
| 距离 5-10mm | +15 |
| 长轴 > 20mm | +20 |
| 长轴 10-20mm | +10 |
| CTR > 0.5 | +10 |

风险判断:
- 总分 < 30: 低风险
- 总分 30-60: 中风险
- 总分 > 60: 高风险
